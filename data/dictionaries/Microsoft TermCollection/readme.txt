What is the structure of the terminology data?

The terminology data from the Microsoft terminology collection is structured as follows in the TBX file:

Term ID
Definition
Source language term
Source language ID
Target language term
Target language ID
The number of terminology entries varies depending on the language and depends on which products have been localized into the respective language.

Wie sieht die Struktur der Terminologiedaten aus? 

Die Terminologiedaten aus der Microsoft-Terminologiesammlung sind in der TBX-Datei folgendermaßen strukturiert:  

Begriffs-ID 
Definition 
Ausgangssprachliche Benennung 
Ausgangssprachen-ID 
Zielsprachliche Benennung 
Zielsprachen-ID 
Die Anzahl der Terminologieeinträge variiert je nach Sprache und ist abhängig davon, welche Produkte in die jeweilige Sprache lokalisiert wurden.
